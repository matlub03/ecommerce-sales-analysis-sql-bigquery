# E-Commerce Sales Data Analysis Using SQL in Google BigQuery

## 📌 Project Overview

This project demonstrates SQL-based data analysis using Google BigQuery on an e-commerce transaction dataset. The objective is to answer 10 business case scenarios related to sales performance, revenue analysis, shipping costs, product performance, customer behavior, and refund analysis.

---

## 🛠 Tools

- Google BigQuery
- SQL
- Google Cloud Platform (GCP)

---

## 📊 Dataset

The dataset contains approximately 10,000 e-commerce transaction records, including:

- Sales Date
- Order ID
- Customer Name
- Product Name
- Category
- Quantity
- Shipping Fee
- Total Sales
- Order Status
- Customer Location

---

## 📈 Business Cases

- Total & Average Shipping Cost
- Top Selling Products
- Product Revenue Analysis
- Quarterly Sales Performance
- Shipping Cost Analysis
- Refund Analysis
- Customer Purchase Analysis
- Pareto Revenue Analysis
- Customer Order Interval
- Revenue Recovery Opportunity

---

## 💻 SQL Skills Demonstrated

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM()
- AVG()
- COUNT()
- COUNT DISTINCT
- CASE WHEN
- COUNTIF()
- SAFE_DIVIDE()
